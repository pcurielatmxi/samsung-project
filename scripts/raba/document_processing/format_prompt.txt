Format the extracted RABA quality test report information into the required JSON structure.

Rules:
- assignment_number: RABA assignment number (e.g., "A22-016871")
- report_date: Format as YYYY-MM-DD
- test_type: Type of test (e.g., "Compressive Strength", "Soil Compaction", "Moisture-Density")
- location: Building + level + grid if available (e.g., "FAB1 L4 D/13-17")
- passed: true if ALL tests passed, false if ANY failed
- failure_reason: Brief explanation if tests failed, null otherwise
- contractor: Contractor name if mentioned, null otherwise
- material_supplier: Material supplier if mentioned, null otherwise
- total_tests: Total number of tests/specimens as integer, null if not specified
- tests_passed: Count of passed tests as integer, null if not specified
- tests_failed: Count of failed tests as integer, null if not specified

Use null for any field where information is not available in the extracted content.
